# Tattoo Design Prompts

## Generate a Design Concept Brief
```
I'm a tattoo artist. My client wants a tattoo with the following details:
- Subject: [e.g. wolf, rose, geometric pattern]
- Style: [e.g. blackwork, Japanese, realism]
- Placement: [e.g. forearm, back, ribs]
- Size: [e.g. palm-sized, half sleeve]
- Mood/feeling they want: [e.g. powerful, delicate, spiritual]

Give me 3 distinct concept directions I can sketch from, with notes on
composition and key visual elements for each.
```

## Turn a Client's Vague Idea Into a Clear Brief
```
My tattoo client described what they want as: "[paste their exact words here]"

Help me turn this into a clear, detailed design brief I can work from.
Include: subject matter, style recommendation, placement suggestion, size,
and any symbolic meaning I should incorporate.
```

## Flash Sheet Concept Generator
```
I'm creating a flash sheet of [number] designs in [style] style.
The theme is [theme e.g. Japanese botanicals / dark surrealism / Southeast Asian traditional].
Give me [number] distinct design ideas that would work well together as a cohesive
flash sheet, with a short description of each.
```

## Adapt a Design for a Different Culture
```
I have a client from [country/culture] who wants a tattoo with [subject/theme].
Help me understand any cultural significance, symbolism, or sensitivities I should
be aware of so I can design something that feels authentic and respectful.
Suggest any traditional visual elements from that culture I could incorporate.
```

## Design Critique and Improvement
```
I've sketched a tattoo design. Here's my description of it: [describe your sketch]
It's going on a [placement] in [style].
Give me honest critique on:
- Composition and balance
- How it will age on skin
- Any elements that might not translate well when tattooed
- Suggestions to make it stronger
```

## Generate Mood Board Directions
```
I need inspiration for a new tattoo project. The theme is [theme].
Give me 5 distinct visual directions I could take this — describe the aesthetic,
colour palette (even for black and grey), reference artists or art movements,
and key visual elements for each direction.
```

## Sticker / Merch Design Brief
```
I want to create [stickers / prints / merch] based on my tattoo art style.
My style is [describe your style].
Help me brainstorm 5 design concepts that would work well as [product type],
appeal to tattoo fans, and be commercially viable.
```
