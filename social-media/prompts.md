# Social Media Prompts

## Write an Instagram Caption for a Finished Tattoo
```
I just finished this tattoo: [describe the tattoo — subject, style, placement, anything interesting about the design or session].
Write 3 Instagram caption options:
1. Short and punchy (under 50 words)
2. Storytelling style (100-150 words)
3. Educational (explains the style or technique)
Include relevant hashtag suggestions for each.
```

## Content Ideas for the Week
```
I'm a tattoo artist based in Thailand known for [your style].
Give me 7 Instagram content ideas for this week — one per day — that mix:
finished work, behind the scenes, educational content, personal brand, and engagement posts.
```

## Write a Bio for My Instagram Profile
```
Write a punchy Instagram bio for a professional tattoo artist with these details:
- Name: Richard Beston
- Based in: Thailand, also works in UK, Germany, Sweden
- Known for: [your style]
- Also: international convention judge, tech-savvy artist
- Booking: [your booking method]
Keep it under 150 characters.
```

## TikTok / Reels Script for a Tattoo Process Video
```
I'm filming a time-lapse/process video of me tattooing [describe the piece].
Write a short voiceover script (30-60 seconds) that:
- Hooks viewers in the first 3 seconds
- Explains what makes this design interesting
- Ends with a call to action (follow, book, comment)
Tone: [e.g. educational / hype / calm and artistic]
```

## Respond to a Negative Comment Professionally
```
Someone left this comment on my Instagram post: "[paste comment]"
Write 2 response options:
1. A calm, professional reply that defuses the situation
2. A witty but non-aggressive reply that shows confidence
```

## Plan a Month of Content Around a Convention
```
I'm attending [convention name] in [city] in [month].
Help me plan 4 weeks of Instagram content building up to, during, and after the convention.
Include: pre-event hype, behind the scenes during, highlights after, and how to convert
convention exposure into new bookings.
```

## Email Campaign to Fill the Booking Calendar
```
I'm a tattoo artist and I want to send an email to my existing client list to fill
my booking calendar for [month/period]. I'm based in [location] during that time.
Write a short, personal email that:
- Feels warm and genuine, not salesy
- Highlights any new styles or availability
- Has a clear call to action to book
- Is under 200 words
```
