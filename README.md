## 💡 How to Use This Library

1. Click any folder or guide link above
2. Find the prompt you need
3. Copy it into [Claude](https://claude.ai) or [Perplexity](https://perplexity.ai)
4. Replace anything in `[brackets]` with your specifics
5. No GitHub account needed to read or download anything here
