# Richard Beston's AI Prompt Library 🎨

A personal collection of Claude & AI prompts for tattoo art, design, client work, and creative workflows.

## Categories

| Folder | What's Inside |
|--------|---------------|
| [tattoo-design](./tattoo-design) | Prompts for generating concepts, flash sheets, and design briefs |
| [client-comms](./client-comms) | Consultation templates, booking replies, pricing explanations |
| [social-media](./social-media) | Caption writing, hashtag research, post ideas |

| [game-dev](./game-dev) | Prompts for Unreal Engine and game design work |

## How to Use

Each file contains ready-to-use prompts. Copy them into Claude and customize the parts in `[brackets]` for your specific needs.
